# 🚌 Kwidzyn Bus

Simple offline Android app with bus schedules for Kwidzyn.

## Features

- 📅 Offline bus schedules
- 🗺 Zoomable route map
- 💳 Ticket prices and discounts
- 📱 Works without internet
- 🎨 Modern Jetpack Compose UI
- 🔍 Expandable timetable cards

## Technologies

- Kotlin
- Jetpack Compose
- Material 3
- JSON assets

## Screens

- Home screen
- Schedule categories
- Bus lines
- Timetables
- Route map
- Ticket prices
